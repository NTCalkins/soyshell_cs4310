Test cases for "mkdir":

1. Successfully creating a new directory.
2. Successfully creating a new directory with one or more spaces in the directory's name.
3. Failing to create a new directory due to another directory having the same name. (Fails atm)
4. Inputting too many arguments. (Fails atm: Returns "No arguments given")
5. Inputting too few arguments.
6. Creating a directory within another directory. (Fails atm and does not prompt the user that it failed)
7. Using ".." within an argument.
